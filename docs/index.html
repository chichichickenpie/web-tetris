1. システム概要

Tetris Guideline 準拠のブラウザ用テトリスゲーム。ビルドツール・パッケージマネージャ不要の単一 HTML ファイル (tetris.html) で構成される。 Google Cloud Platform 上の複数サービスにデプロイされ、静的コンテンツとして配信される。

3. アプリケーションアーキテクチャ

全ロジックは tetris.html 内の <script> タグに収められ、6 つのクラスで構成される。

classDiagram class Bag { +queue: string[] +next() string -shuffle() } class Piece { +type: string +rot: number +x: number +y: number +cells() number[][] } class Board { +grid: number[][] +collision(piece) boolean +lock(piece) +clearLines() number } class Game { +state: START|PLAYING|PAUSED|GAME_OVER +score: number +level: number +lines: number +update(delta, softDrop) -detectTSpin() boolean } class Renderer { +drawBoard() +drawHold() +drawNext() } class Input { +update(delta) -handleDAS() -handleARR() } Game --> Board Game --> Bag Game --> Piece Renderer --> Board Renderer --> Game Input --> Game

6. Tetris Guideline 準拠機能

機能実装方式
ランダマイザ7-bag（全 7 ピースをシャッフルして順に取り出す）
回転システムSRS（Super Rotation System）
壁蹴りJLSTZ 用テーブル / I ピース専用テーブルを分離
T-Spin 判定T バウンディングボックス 4 隅のうち 3 隅以上が埋まっていれば T-Spin
Back-to-BackTetris / T-Spin 連続時に 1.5× スコア倍率
ゴーストピース落下先を半透明で表示
ホールド1 ピースを保留、次のピースと交換可能
NEXT キュー次の 5 ピースを表示
